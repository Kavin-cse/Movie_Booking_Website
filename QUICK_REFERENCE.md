## 💰 Pricing Reference

### Seat Categories
| Category | Price | Location |
|----------|-------|----------|
| VIP | ₹250 | Front rows (VIP section) |
| Premium | ₹200 | Middle rows |
| Regular | ₹150 | Back rows |

### Calculations
- **Subtotal**: Sum of all selected seat prices
- **Tax**: 18% GST on subtotal
- **Discount**: ₹50 per seat if subtotal > ₹1000
- **Final Amount**: Subtotal + Tax - Discount

### Example Pricing
```
3 VIP seats: 3 × ₹250 = ₹750
Subtotal: ₹750
Tax (18%): ₹135
Discount: ₹0 (< ₹1000)
Total: ₹885
```

```
8 seats mix: ₹1,550
Tax (18%): ₹279
Discount (₹50×8): -₹400
Total: ₹1,429
```

## 📋 Data Model Structure

```
Movie {
  id: number
  title: string
  genre: string
  rating: number
  duration: number
  language: string
  showTimes: ShowTime[]
}

ShowTime {
  id: string
  time: string
  format: string
  language: string
  availableSeats: number
}

Seat {
  id: string
  seatNo: string
  row: string
  column: number
  isAvailable: boolean
  isSelected: boolean
  price: number
  category: 'regular' | 'premium' | 'vip'
}

Booking {
  id: string
  movieId: number
  showTimeId: string
  selectedSeats: Seat[]
  totalPrice: number
  customerEmail: string
  customerPhone: string
  status: 'confirmed' | 'pending' | 'cancelled'
  paymentMethod: 'card' | 'upi' | 'wallet'
}
```
