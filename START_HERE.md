## 🛠️ TECHNICAL ARCHITECTURE

### Component Structure
```
AppComponent (Root)
├── MovieListComponent          ← Page 1: Browse Movies
├── MovieDetailComponent        ← Page 2: Movie Info
├── BookingComponent            ← Page 3: Seat Selection
├── PaymentComponent            ← Page 4: Payment Form
└── ConfirmationComponent       ← Page 5: Confirmation
```

### Service Layer
```
MovieService
├── getMovies()                 → Get all movies
├── getMovieById()              → Get single movie
└── getByGenre()                → Filter by genre

SeatService
├── initializeSeats()           → Generate seat grid
├── getSeats()                  → Get current seats
├── selectSeat()                → Toggle seat selection
└── resetSeats()                → Clear selection

BookingService
├── createBooking()             → Create new booking
├── getBookings()               → Fetch all bookings
├── getBookingById()            → Get single booking
└── cancelBooking()             → Cancel booking
```

### State Management
```
Local State (Signals)
├── movies:signal              → Movie list
├── selectedGenre:signal       → Active filter
├── selectedSeats:computed     → Selected seats
└── totalPrice:computed        → Calculated price

Shared State (BehaviorSubject)
├── seatsSubject               → Seat grid state
├── bookingsSubject            → Booking history
└── Router State               → Navigation data
```
